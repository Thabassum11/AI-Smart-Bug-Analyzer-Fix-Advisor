# AI Smart Bug Analyzer & Fix Advisor

## Defect Analysis Workflow

1. Bug Identification
2. Bug Reporting
3. Bug Triage
4. Root Cause Analysis
5. Bug Fixing
6. Testing
7. Closure

## RAG Architecture

RAG stands for Retrieval Augmented Generation.

The system retrieves similar historical bug reports and uses them to generate accurate recommendations.

## Semantic Similarity

Semantic similarity identifies bug reports that have similar meaning even if different words are used.

## Bug Report Structure

- Title
- Description
- Steps to Reproduce
- Expected Result
- Actual Result
- Severity
- Logs
- Stack Trace
